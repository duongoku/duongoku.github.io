How to use it

Paste the raw sample values exactly as exported from a spreadsheet or grade book.
Input rules
  • Accepts line breaks, spaces, commas, and semicolons.
  • Ignores blank entries and reports invalid tokens.
  • Uses every valid number for summary statistics.
  • Plots values inside the selected axis range.

Reading the chart

The chart combines exact bin counts with a smoothed distribution shape.
Visual marks
  • Bars show how many samples fall in each score band.
  • The curve shows the overall distribution shape.
  • The solid vertical line marks the mean.
  • The dashed vertical line marks the median.

Interpretation note

  • For class scores, use the same axis range for every section before comparing plots.
  • Very small samples can make the smooth curve look more precise than the data really is.
  • Zeros are included because they often represent meaningful class outcomes.
